## 🚀 Como publicar no GitHub Pages

### 1. Criar conta e repositório

1. Acesse [github.com](https://github.com) e crie uma conta grátis
2. Clique em **"New repository"**
3. Nome: `glowup-feminino` → deixe **Public** → **Create repository**

### 2. Fazer upload dos arquivos

1. Dentro do repositório, clique em **"uploading an existing file"**
2. Arraste os 3 arquivos: `quiz.html`, `index.html`, `README.md`
3. Clique em **"Commit changes"**

### 3. Ativar GitHub Pages

1. Clique em **Settings** (engrenagem)
2. Menu lateral → **Pages**
3. Branch: `main` · Pasta: `/ (root)` → **Save**
4. Aguarde ~2 minutos ☕

### 4. Seus links ficam assim

| Página | Link |
|--------|------|
| **Quiz (divulgue este)** | `https://SEU-USUARIO.github.io/glowup-feminino/quiz.html` |
| Página de vendas | `https://SEU-USUARIO.github.io/glowup-feminino/` |

> Substitua `SEU-USUARIO` pelo seu usuário do GitHub.
